# 行動規範 (Code of Conduct)

## 私たちの誓い

私たち貢献者およびメンテナーは、オープンで歓迎されるコミュニティを育成するために、年齢、身体の大きさ、障害、民族、性自認および表現、経験レベル、教育、社会経済的地位、国籍、外見、人種、宗教、または性的アイデンティティや指向に関わらず、すべての人にとってハラスメントのない参加体験を提供することを誓います。

## 私たちの基準

コミュニティにポジティブな環境をもたらす行動の例：

* 歓迎的で包括的な言葉遣いをする
* 他の人の視点や経験を尊重する
* 建設的な批判を丁重に受け入れる
* コミュニティにとって最善であることに焦点を当てる
* 他のコミュニティメンバーに対して共感を示す

参加者による許容されない行動の例：

* 性的な言葉や画像の不適切な使用、および歓迎されない性的注目や求愛
* トローリング、侮辱的・軽蔑的なコメント、個人的または政治的な攻撃
* 公的または私的な嫌がらせ
* 許可なく他人の個人情報（住所、電子メールアドレスなど）を公開すること
* その他、専門的な環境において不適切と見なされる行為

## 適用範囲

この行動規範は、コミュニティ内だけでなく、コミュニティを代表している場合（公的なメールアドレスの使用、公式ソーシャルメディアアカウントへの投稿、イベントでの代表など）の公的スペースにも適用されます。

## 執行

不適切、ハラスメント、またはその他の不快な行為は、プロジェクトチーム（[dexism1024@gmail.com](mailto:dexism1024@gmail.com)）に報告してください。プロジェクトチームは、報告された事案を公平かつ客観的に調査し、コミュニティの安全を確保するために必要かつ適切と判断される対応を行います。
