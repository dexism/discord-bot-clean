# セキュリティポリシー (Security Policy)

## サポート対象バージョン

以下のバージョンのセキュリティアップデートのみをサポートしています。

| バージョン | サポート状況 |
| :-: | :-: |
| 3.10.x | :white_check_mark: |
| < 3.10 | :x: |

## 脆弱性の報告

脆弱性を発見した場合は、Issueには**絶対に書き込まず**、開発チームへ直接ご連絡ください。
Discordサーバーの管理者、または `internal-docs/operations.md` に記載の連絡先へご一報をお願いいたします。

報告受理後、チーム内で調査を行い、修正が必要と判断された場合は修正パッチを含むアップデートをリリースします。
報告者への感謝の意を表し、可能であればリリースノートにてクレジット表記させていただきます。
